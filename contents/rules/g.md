# /g/ Rules

/g/ is for technology, software, hardware, programming, systems, hosting, tools, and adjacent
technical culture.

## On Topic

- Programming help, code review, tooling, operating systems, servers, networking, and hardware are welcome.
- Project threads are welcome when they show actual work, screenshots, code, logs, or a concrete question.
- Security discussion is allowed when it is educational or defensive.
- Low-effort brand wars and repeated platform bait may be removed.

## Not Allowed

- Requests for malware, credential theft, botnets, account compromise, or evasion of another service's rules.
- Posting private keys, tokens, passwords, leaked databases, or personal information.
- "Hack this for me" threads, revenge requests, or social engineering scripts.
- AI slop dumps with no question, context, or original effort.

## Good Threads

- Include the exact error, environment, command, or browser when asking for help.
- Say what you tried already.
- Link documentation or source code when relevant.
- If you solved it, post the fix so the thread becomes useful later.

## Uploads

- Screenshots, logs, diagrams, benchmark images, and setup photos are fine when they add context.
- Redact tokens, private hostnames, private IPs, customer data, and personal paths before posting.
- Do not post suspicious binaries, loaders, cracked tools, or "download and run this" download links.
