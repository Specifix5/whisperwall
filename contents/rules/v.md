# /v/ Rules

/v/ is for video games: playing them, making them, collecting them, speedrunning them, and
complaining about patch notes with some dignity.

## On Topic

- Game discussion, hardware, emulation, speedruns, mods, game dev, preservation, and recommendations are welcome.
- Spoilers are allowed only when clearly marked.
- Technical help is fine when it is about games, launchers, controllers, saves, or performance.

## Not Allowed

- Console-war spam, repeated bait, or harassment of developers and players.
- Unmarked major spoilers in active discussion threads.

## Good Threads

- Put the game title in the subject.
- Mark spoilers for spoilers
- For troubleshooting, include platform, version, GPU/CPU when relevant, and the exact error.
